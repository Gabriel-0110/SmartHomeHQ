# 00-Inventory

## Overview

This folder contains comprehensive inventory and tracking information for all smart home devices and hardware components.

## Contents

- **device-inventory.csv**: Master inventory of all smart home devices
- **purchase-records.md**: Purchase history and receipts
- **warranty-info.md**: Warranty information and RMA procedures
- **hardware-specs.md**: Detailed hardware specifications

## Device Inventory

The `device-inventory.csv` file tracks:
- Device name and category
- Manufacturer and model information
- Physical location
- Purchase details (date, price)
- Network information (IP, MAC address)
- Current status and notes
- Warranty expiration dates

### Categories

- **Hubs**: Home Assistant, Homebridge, Scrypted servers
- **Lights**: Smart bulbs, switches, strips
- **Sensors**: Motion, temperature, door/window, leak sensors
- **Cameras**: Security cameras, doorbells
- **Climate**: Thermostats, fans, air quality monitors
- **Locks**: Smart locks and access control
- **Entertainment**: TVs, speakers, streaming devices
- **Appliances**: Smart plugs, appliances, vacuum robots
- **Network**: Routers, switches, access points

## Maintenance

- Update inventory when adding or removing devices
- Record all purchases and warranty information
- Track firmware versions for critical devices
- Note any device issues or RMA history

## File Formats

- Use CSV for inventory to enable easy import/export
- Use Markdown for detailed specifications and notes
- Include photos/diagrams in an `assets/` subfolder if needed
